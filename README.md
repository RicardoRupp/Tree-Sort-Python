# Tree Sort em Python

## Descrição
Este projeto implementa o algoritmo **Tree Sort** utilizando uma **Árvore Binária de Busca (BST)**.

O algoritmo insere os elementos em uma árvore binária e depois realiza um percurso em ordem (*inorder traversal*) para produzir os dados ordenados.

## Funcionamento
1. Insere os elementos na árvore.
2. Organiza automaticamente conforme as regras da BST.
3. Percorre a árvore em ordem.
4. Exibe os elementos ordenados.

## Estrutura do código

- `Node`
  - Representa os nós da árvore.

- `insert()`
  - Insere novos elementos.

- `insertRec()`
  - Inserção recursiva.

- `inorderRec()`
  - Percurso em ordem.

- `treeins(arr)`
  - Insere todos os elementos.
